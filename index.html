Multi Agent Systems

Le système multi-agent en Java est composé d'un Main Container, qui organise et supervise la communication entre trois agents. L'Agent Fibonacci génère une suite de Fibonacci, tandis que l'Agent Carré produit une suite de nombres carrés. Ces deux séquences sont ensuite envoyées à l'Agent AgClass, qui fusionne les deux suites et les trie en ordre croissant, assurant ainsi une gestion efficace des données générées par les agents.
